第1回 WWW と HTML

今回の授業で学んだことは、HTMLの要素や属性、文章構成について学びました。 HTMLでは、見出し、段落、リンク、画像などを定義する方法を習得しました。これにより、基本的なウェブページの作成と、ブラウザがHTML文書をどのように解釈して表示するかを理解することができました。

演習の成果:演習 1-4: HTMLのさまざまな要素

第2回 CSS による WEB ページの装飾

CSSは、HTMLなどのマークアップ言語で書かれたWebページの見た目やレイアウトを制御するスタイルシート言語です。CSSを使用すると、フォント、色、間隔、配置、アニメーションなどのビジュアル要素を効率的に設定し自分の好みに合わせたでサインをすることができる。

演習の成果:練習問題2-3

第3回 JavaScript の基本

JavaScriptは、プログラムを通じて HTML 要素や属性,CSS などの追加・編集・削除を可能にしてくれます.また、javaとjavascriptでは少し書き方が違うので注意して書きます.

演習の成果:練習問題3-6

第4回 DOM と Javascript

JavaScriptのDOM操作は、ウェブページのHTML要素を動的に変更する方法です。要素の検索をしたり、テキストの変更,CSSを変更や新しい要素の追加、そして要素の削除などできることがたくさんあります。

演習の成果:練習問題4-2から練習問題4-5

第5回 イベント処理

クリックというイベントを実行したら見出しの文字を変更したり、背景色をランダムに変更したりするプログラムをHTMLやjacascriptを駆使して作った。javascriptではイベントハンドラの登録や定義を行いHTMLではボタンを配置する設定を施しています。

演習の成果:練習問題5-3
