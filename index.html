「WEBコンテンツ作成」の成果

本テーマのついて

Webコンテンツ作成の授業を通して私は、Webコンテンツについての基本的な理解と実践的なスキルを身につけることです。 目標は、HTML、CSS、JavaScriptなどの技術を用いて、魅力的で機能的なウェブサイトを構築できるようになることです。

プロジェクト課題について

私は、世界の天気についてのプロジェクト課題について作成しました。今回の全体の授業を通してHTMLやCSS、Javascriptの使い方やイベント処理、通信処理を学びそれを全て駆使してプロジェクト課題を作りました。イベント処理では、プルダウンメニューを使い全ての国の天気が検索しやすいようにし、全体的に見やすいようにしました。 また、CSSではふんわりするような柔らかい色合いの雰囲気の感じで作成しました。

プロジェクト課題で作成したページ:project.html

数当てゲーム

javascriptやHTMLを使って数当てゲームを作成しました。 乱数で1~10の値をランダムで1つ決めてそれを答えとする。答える人は答えを予想する。ただし、予想する回数は最大で3回となりそれ以降は答えを教えてくれるようにする。 また、答えた人の予想が違かった場合、入力した値よりも答えが大きければ「まちがい.答えはもっと大きいですよ」入力した値よりも小さければ「まちがい.答えはもっと小さいですよ」と出力できるようにする。 3回以内に答えの数を当てれば勝ち、そうでなければ負けとなる。またページには、テキストボックスとボタンを設置するようにしてそこに答える人は1~10の値を入力する。回答というボタンを押せばプログラムが起動して実際にゲームが始まります。

数当てゲームのページ:kazuate.html

第1回 WWW と HTML

今回の授業で学んだことは、HTMLの要素や属性、文章構成について学びました。 HTMLでは、見出し、段落、リンク、画像などを定義する方法を習得しました。これにより、基本的なウェブページの作成と、ブラウザがHTML文書をどのように解釈して表示するかを理解することができました。

演習の成果:演習 1-4: HTMLのさまざまな要素

第2回 CSS による WEB ページの装飾

CSSは、HTMLなどのマークアップ言語で書かれたWebページの見た目やレイアウトを制御するスタイルシート言語です。CSSを使用すると、フォント、色、間隔、配置、アニメーションなどのビジュアル要素を効率的に設定し自分の好みに合わせたでサインをすることができる。

演習の成果:練習問題2-3

第3回 JavaScript の基本

JavaScriptはWebページに動的な機能を追加するためのプログラミング言語です。またJavaScriptは、プログラムを通じて HTML 要素や属性,CSS などの追加・編集・削除を可能にしてくれます.また、javaとjavascriptでは少し書き方が違うので注意して書きます.

演習の成果:練習問題3-6

第4回 DOM と Javascript

JavaScriptのDOM操作は、ウェブページのHTML要素を動的に変更する方法です。要素の検索をしたり、テキストの変更,CSSを変更や新しい要素の追加、そして要素の削除などできることがたくさんあります。

演習の成果:練習問題4-2から練習問題4-5

第5回 イベント処理

クリックというイベントを実行したら見出しの文字を変更したり、背景色をランダムに変更したりするプログラムをHTMLやjacascriptを駆使して作った。javascriptではイベントハンドラの登録や定義を行いHTMLではボタンを配置する設定を施しています。

演習の成果:練習問題5-3
